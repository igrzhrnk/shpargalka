Список команд git

git checkout -b gh-pages
Команда для створення гілки. В даномму випадку це гілка gh-pages.
git add .
Додавання файлів до трекінгу.
git commit -m "some commit"
Створення нового коміта.
git push origin gh-pages
Відправка коду на github.

Список команд npm

npm install package_name
Встановлення пакета package_name локального.
npm install -g package_name
Встановлення пакета package_name глобально.
npm install package_name -save
Встановлює пакет локально і створює посилання на пакет в файлі package.json.
npm install package_name -save-dev
Встановлює пакет локально і створює посилання на пакет в файлі package.json якщо він потрібний тільки для розробки.
npm uninstall package_name
Видаляє локальний пакет package_name.
npm uninstall -g package_name
Видаляє глобальний пакет package_name.
npm uninstall -save (or --save-dev) package_name
Для видалення локального пакета разом з видаленням посилання на нього з файла package.json..
npm outdated
Виводить список застарілих пакетів з зазначенням номерів версій.
npm update
Оновлює всі локальні пакети.
npm update package_name
Оновлює локальний пакет package_name.
npm update -g
Оновлює всі глобальні пакети.
npm update -g package_name
Оновлює глобальний пакет package_name.
